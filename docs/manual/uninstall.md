# Uninstall
> How to uninstall the extension if you don't want it anymore


## Extension pane

1. Open the _Extensions_ tab in VS Code.
1. Click the Auto Commit Message extension.
1. Click _uninstall_.
1. Restart VS Code.

Or

## Command-prompt

1. Run this command.
    ```sh
    $ code --uninstall-extension MichaelCurrin.auto-commit-msg
    ```
2. Restart VS Code.
